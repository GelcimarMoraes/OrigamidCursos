Pontuação

-

Hífen
(Hyphen)

  • Mac -
  • Win -
  • HTML -

Ligar elementos de palavras compostas e unir pronomes átonos a verbos.

Coca-Cola. Vou buscá-la.

–

Meia-risca
(En Dash)

  • Mac option + -
  • Win alt + 0150
  • HTML &ndash;

Indica distância entre um número e outro. Ou a relacão entre duas coisas.

Vitaminas de A–Z. A viagem Rio–São Paulo

—

Travessão
(Em Dash)

  • Mac option + shift + -
  • Win alt + 0151
  • HTML &mdash;

Indicar a fala da personagem. Separar frases explicativas.

Jon — um grande amigo da família — foi visto hoje cedo.

“

Aspas (Abre)
(Opening Quotes)

  • Mac option + [
  • Win alt + 0147
  • HTML &ldquo;
”

Aspas (Fecha)
(Closing Quotes)

  • Mac option + shift + [
  • Win alt + 0148
  • HTML &rdquo;

Utilizada em citações, para representar falas, destacar palavras e etc.
Cuidado com as aspas burras isso "" é diferente disso “”

“Há algo no uivar do lobo que tira um homem do aqui e agora e o transporta para uma floresta da mente.”
